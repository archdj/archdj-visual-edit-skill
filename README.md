# visual-edit

프론트엔드 UI를 스크린샷으로 찍어서 빨간 박스/화살표로 "이 부분 수정"이라고 표시하면, Claude가 해당 영역만 정확히 고쳐주는 시각적 피드백 루프 스킬.

## 설치

```bash
npx skills@latest add archdj/archdj-visual-edit-skill
```

## 사용법

1. 프론트엔드 코드 작성 후 브라우저에서 미리보기
2. 스크린샷 찍어서 수정할 부분에 박스/화살표/메모 표시
3. Claude Code에 이미지 붙여넣기
4. Claude가 해당 영역만 수정

## 트리거

"미리보기", "스크린샷", "이 부분 수정", "영역 수정" 언급 시 또는 UI 이미지에 그림이 그려져 있을 때 자동 작동.
